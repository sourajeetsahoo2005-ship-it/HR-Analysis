# HR Analysis Dashboard

## Description
HR Analytics Dashboard project developed in Microsoft Excel to analyze employee attrition, performance, department-wise trends, gender distribution, and workforce KPIs using interactive dashboards, pivot tables, and charts. This project helps in identifying workforce patterns and supports data-driven HR decision-making through clear visualizations and reports.

## Features
- Employee Attrition Analysis
- Department-wise Performance Insights
- Gender & Age Distribution Analysis
- Interactive Dashboard with Filters
- KPI Tracking and Visualization
- Pivot Tables & Charts for Data Analysis
- Workforce Trend Analysis
- Easy-to-understand HR Reports

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
